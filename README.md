# ⚽ FIFA World Cup Qatar 2022

A stunning, responsive FIFA World Cup website with live standings, match results, and player profiles.

## Features

- 📊 **Group Standings** — All 8 groups with real tournament data
- ⚽ **Match Results** — 48 matches with scores, venues, and matchdays
- 🏅 **301 Player Profiles** — Full squads with team-colored avatars
- 🎨 **Animations** — Scroll-triggered, hover effects, animated counters
- 📱 **Fully Responsive** — Works on desktop, tablet, and mobile
- 🔄 **Auto-refresh** — Data refreshes every 24 hours

## Tech Stack

- **Frontend:** React + Vite + Tailwind CSS + shadcn/ui
- **Backend:** Hono (Bun) + Prisma + SQLite
